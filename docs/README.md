# Documentation Hub

This directory is the central repository for all technical knowledge and institutional memory for the Uptime Monitor.

The Hub is documented as a closed-loop monitoring system: source of truth, runtime, signals, decisions, actions, and memory.

## Documentation Domains

| Domain | Description |
| --- | --- |
| [Architecture](architecture/README.md) | System design, boundaries, data flow, and deployment shape. |
| [Decisions](decisions/README.md) | Architecture decision records and major technical tradeoffs. |
| [Incidents](incidents/README.md) | Incident notes, RCAs, operational learnings, and follow-up actions. |
